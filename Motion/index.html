googleVis motion chart of Response Burden Data

This is a motion chart where you can view the interaction between forms issued, forms returned, and response rate over time for CSO biennial, annual, quarterly, monthly and once off surveys. There are options to focus on individual, multipe or all surveys as well as to change the way in which the output is visualised. The information for the chart is taken from the Response Burden Barometer data on the CSO statbank. Response rate is not on the CSO statbank. It is calulated by dividing forms returned by forms issued for a particular survey.

require("googleVis")
require("pxR")
require("reshape2") 
RBA01.px<-read.px("http://www.cso.ie/px/pxeirestat/Database/eirestat/Response%20Burden%20Barometer/RBA01.px")
RBA01.df<-as.data.frame(RBA01.px)

RBA01 = dcast(RBA01.df, Year + Survey ~ Statistic, value.var="value", fun.aggregate=sum)

RBA01[,5]<-(RBA01[4]/RBA01[3])
RBB<-cbind(RBA01[1:4],RBA01[,5])
names(RBB) <- c("Year","Survey","Issued", "Returned", "Response.Rate") 
date1 <- as.character(RBB$Year)

strDates <- date1
dates <- as.Date(strDates, "%Y") 
date=as.numeric(format(dates,"%Y"))
RBB$Year= date

chart=gvisMotionChart(RBB, 
                idvar= "Survey", 
                timevar= "Year",
                xvar="Year" ,                 
                yvar="Response.Rate",                      options=list(height=600,width=600)
                             )
plot(chart)
