This Cork based website, which is hosted by github, is a platform for data analytics that focuses initially on twitter api and Central Statistics Office data. In some cases the underlying R code is provided which facilitates replication of the analysis. At a later date the code for the Shiny apps will also be made available online. It is planned to broaden the scope of the analytics over time.
